## 8 if 语句

- 作用：根据条件【有选择性】的执行某些语句-【判断】

- 语法

  ```python
  if 条件1:
      语句1
  elif 条件2:
      语句2
  ...
  else:
      语句n
  ```

- 说明：

  - 1 当if后的条件成立（True）,则执行下面的语句【缩进4个空格】
  - 2 if 后的条件不成立，判断elif后的条件是否成立，如果成立，则执行下面的语句
  - 3 依次向下判断条件是否成立
  - 4 else语句是以上所有的条件都不成立则执行。
